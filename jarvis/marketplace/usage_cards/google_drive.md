Use the google_drive tool to list, search, read, create, share and delete files in Google Drive.
- The full 'drive' scope grants access to ALL the user's files (read/write/delete), not just app-created ones.
- Search by name or MIME type before creating; avoid duplicates.
- Create and upload files directly (full autonomy); report file name and Drive URL afterwards.
- Folders can be created and used as targets for new files.
- Sharing exposes data outward and deleting is destructive — both confirm before acting.
