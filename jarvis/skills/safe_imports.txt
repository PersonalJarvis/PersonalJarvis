# Allowlist erlaubter Python-Imports in Sub-Jarvis-authored Skills
# (Plan-§7.5 Sicherheits-Lint vor Promote, ergänzt durch Phase-7.5-Implementation).
#
# Skills sind Markdown mit YAML-Frontmatter — Python-Code-Blöcke darin
# werden nicht direkt ausgeführt; sie können aber via `scripts/`-Resources
# importiert werden. Dieser Lint scannt jeden Python-Block per AST und
# verbietet alles, was außerhalb dieser Liste liegt.
#
# Format: ein Modulname pro Zeile, # für Kommentare. Submodule werden
# automatisch erlaubt (z.B. `pathlib.Path` → `pathlib` in der Liste).
