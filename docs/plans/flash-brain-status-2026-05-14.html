BUG-019 · BUG-017 cascade

Force-spawn-openclaw hangs when brain.primary is non-Claude/non-Gemini

  • Symptom. User says "such mir Flüge nach München" . Flash-Brain ack plays correctly, then Jarvis stays silent — no main answer. User words: "er hört nur noch zu, denkt nicht mehr" . Repeats on every action-verb utterance.
  • Fix. jarvis/brain/manager.py::_should_force_openclaw now returns False when brain.primary is not "claude-api" or "gemini". Router-Brain handles the request inline via the normal tool-use loop instead of fire-and-forgetting into a dead GeminiWorker that hits the BUG-017 403. Ships in commit 4fd8932b.
  • Cross-ref. Root cause BUG-017 (Gemini account 403). Complementary TTS-pipe fix in commit 8b0d7427 keeps the Flash-Brain ack audible on the inline path. Both gates documented in ADR-0014 and full lessons in docs/BUGS.md#BUG-019.
