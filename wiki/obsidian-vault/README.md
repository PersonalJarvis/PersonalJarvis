# Obsidian Vault — Personal Jarvis Wiki

Open this folder with **Obsidian → Open folder as vault** (point at `wiki/obsidian-vault/`); the dark *Minimal* theme and the enabled core plugins will load automatically from the committed `.obsidian/` configuration. New notes go into `10-notes/` (or `00-index/` if they are landing pages); the `99-templates/` folder is wired up as the templates source. Cross-link notes by typing `[[` and choosing a target — every Wikilink becomes a navigable edge in the graph view (**Ctrl+G**).
